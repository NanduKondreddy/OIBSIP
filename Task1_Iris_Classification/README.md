# Iris Flower Classification

This project uses machine learning to classify iris flowers into Setosa,
Versicolor, and Virginica species based on their sepal and petal measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model
- Logistic Regression

## Result
The model achieves high accuracy in classifying iris flower species.
